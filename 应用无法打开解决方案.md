# 应用无法打开的解决方案

如果 `录屏软件.app` 无法打开，这是 macOS 的安全机制（Gatekeeper）在阻止未签名的应用。

## 快速解决方案

### 方法 1: 右键打开（最简单）

1. 找到 `dist/录屏软件.app`
2. **右键点击**应用图标
3. 选择**"打开"**（不是双击）
4. 会弹出安全提示，点击**"打开"**
5. 之后就可以正常双击打开了

### 方法 2: 使用修复脚本

```bash
./fix-app.sh
```

### 方法 3: 在系统设置中允许

1. 打开"系统设置"（System Settings）
2. 进入"隐私与安全性"（Privacy & Security）
3. 在"安全性"部分，找到被阻止的应用
4. 点击"仍要打开"（Open Anyway）

### 方法 4: 使用命令行移除隔离属性

```bash
xattr -cr "dist/录屏软件.app"
```

然后右键点击应用选择"打开"。

## 为什么会出现这个问题？

macOS 的 Gatekeeper 会检查应用的代码签名。未签名的应用（或使用 ad-hoc 签名的应用）会被阻止运行，需要用户手动允许。

## 永久解决方案（需要开发者账号）

如果你有 Apple 开发者账号，可以对应用进行代码签名：

```bash
codesign --deep --force --verify --verbose --sign "Developer ID Application: Your Name" "dist/录屏软件.app"
```

然后进行公证（notarize）：

```bash
xcrun notarytool submit "dist/录屏软件.app" --keychain-profile "your-profile" --wait
```

## 分发给其他用户

如果要将应用分发给其他用户，建议：

1. **压缩为 zip 文件**：macOS 会自动为下载的文件添加隔离属性
2. **提供说明**：告诉用户首次运行时需要右键打开
3. **或者进行代码签名**：使用开发者账号签名后，用户可以直接打开

## 常见错误

**错误**: "录屏软件.app 已损坏，无法打开"

**解决**: 这不是真的损坏，只是 macOS 的安全提示。使用上面的方法 1 或方法 4 即可。
