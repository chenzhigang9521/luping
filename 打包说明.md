# 打包说明 - macOS 和 Windows 版本

## ✅ macOS 版本（已完成）

**位置**: `dist/录屏软件.app`

**使用方法**:
1. 双击 `录屏软件.app` 即可运行
2. 首次运行需要在"系统设置 > 隐私与安全性 > 屏幕录制"中授权
3. 可以将整个 `.app` 文件分发给其他 macOS 用户

## 📦 Windows 版本打包方法

由于 PyInstaller 不支持交叉编译，需要在 Windows 系统上打包。有以下几种方法：

### 方法 1: 在 Windows 机器上打包（最简单）

1. 将项目代码复制到 Windows 机器
2. 安装 Rye（如果还没有）:
   ```powershell
   irm https://rye-up.com/get | iex
   ```

3. 同步依赖:
   ```cmd
   rye sync
   ```

4. 运行打包脚本:
   ```cmd
   build-windows.bat
   ```

5. 打包完成后，`dist/录屏软件.exe` 就是 Windows 版本

### 方法 2: 使用 GitHub Actions（推荐，自动化）

如果你将代码推送到 GitHub：

1. 推送代码到 GitHub
2. 创建并推送一个标签:
   ```bash
   git tag v1.0.0
   git push origin v1.0.0
   ```
3. GitHub Actions 会自动构建 macOS 和 Windows 版本
4. 在 GitHub 仓库的 Actions 页面下载构建产物

### 方法 3: 使用虚拟机

在 macOS 上运行 Windows 虚拟机（Parallels、VMware 等），然后在虚拟机中打包。
